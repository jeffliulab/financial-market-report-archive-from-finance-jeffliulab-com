# Financial Market Report Archive

Daily market insight reports auto-generated by [finance.jeffliulab.com](https://finance.jeffliulab.com).

## Structure

```
en/
  YYYY/
    YYYY-MM-DD.md   # English report
zh/
  YYYY/
    YYYY-MM-DD.md   # Chinese report
```

Each pair covers a single trading day's global market activity (US/global indices, sectors, mega-cap stocks, bonds, FX, commodities) plus geopolitics/news synthesis, generated via a 3-phase LLM pipeline (DeepSeek).

## Generation

- **Source**: `finance.jeffliulab.com` repo, `scripts/generate-report.mjs`
- **Schedule**: Daily at 08:00 UTC (systemd timer)
- **Push to this archive**: Automatic, immediately after each generation

## Provenance

This repo is included as a git submodule at `content/market-reports/` of the source project. The website serves reports directly from local disk (Next.js `force-dynamic`), so the live site and this archive stay in sync but are independent — push failures here do not affect the live site.

Initial import: 2026-03-31 through 2026-04-21 (44 files, 22 trading days). Originally stored as flat `YYYY-MM-DD.{en,zh}.md`; reorganized into language/year folders on 2026-04-22.
