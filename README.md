# Financial Market Report Archive

Daily market insight reports auto-generated by [only-what-matters.jeffliulab.com](https://only-what-matters.jeffliulab.com/) (the web companion to the iOS app **Only What Matters**, `com.jeffliulab.only-what-matters`).

> **2026-04-30**: generation moved here from `finance.jeffliulab.com`. The archive
> repo URL, directory layout, and freshness-monitor workflow are unchanged, so
> downstream consumers (notably the iOS app reading raw markdown) are unaffected.

## Structure

```
en/
  YYYY/
    YYYY-MM-DD.md   # English report
zh/
  YYYY/
    YYYY-MM-DD.md   # Chinese report
```

Each pair covers a single trading day's global market activity (US/global indices, sectors, mega-cap stocks, bonds, FX, commodities) plus geopolitics/news synthesis, generated via a 3-phase LLM pipeline (DeepSeek).

## Generation

- **Source**: `only-what-matters.jeffliulab.com` repo, `scripts/generate-report.mjs` (was `finance.jeffliulab.com` before 2026-04-30)
- **Schedule**: Daily at 08:00 UTC + jitter (systemd timer `only-what-matters-daily-report.timer`)
- **Push to this archive**: Automatic, immediately after each generation

## Provenance

This repo is included as a git submodule at `content/market-reports/` of the source project. The website serves reports directly from local disk (Next.js `force-dynamic`), so the live site and this archive stay in sync but are independent — push failures here do not affect the live site.

The iOS app `com.jeffliulab.only-what-matters` reads markdown directly from this repo's GitHub raw URLs, so the archive's URL + directory layout (`{en|zh}/{YYYY}/{YYYY-MM-DD}.md`) is part of the app's contract — do not rename or restructure without coordinating with the app side.

Initial import: 2026-03-31 through 2026-04-21 (44 files, 22 trading days). Originally stored as flat `YYYY-MM-DD.{en,zh}.md`; reorganized into language/year folders on 2026-04-22.
